# 🌐 Portfolio Personnel — Lauclass Ekumu

---

## 📌 Description du projet

Ce projet est une **page de profil personnel (portfolio)** réalisée en **HTML5 pur**, sans CSS ni JavaScript.

Il a été conçu dans le cadre d’un apprentissage des bases du développement web afin de maîtriser :

- la structure d’une page HTML5
- les balises sémantiques
- l’accessibilité web
- la création de formulaires
- l’organisation d’un projet web professionnel
- l’utilisation de Git et GitHub

---

## 🎯 Objectifs pédagogiques

Ce projet permet de développer les compétences suivantes :

- Structurer une page web avec HTML5
- Utiliser correctement les balises sémantiques
- Organiser un contenu professionnel et lisible
- Créer un formulaire HTML accessible
- Comprendre la hiérarchie des sections d’une page web
- Appliquer les bonnes pratiques de développement web
- Versionner un projet avec Git et GitHub
